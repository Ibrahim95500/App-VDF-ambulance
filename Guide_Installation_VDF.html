Guide d'Installation de l'Application Mobile

Installation pour iPhone (iOS)

Uniquement via l'application sécurisée TestFlight d'Apple.

1

Télécharger TestFlight

Ouvrez l'App Store sur votre iPhone, cherchez TestFlight et installez l'application (elle est officielle et certifiée par Apple).

2

Scanner le QR Code

Ouvrez l'appareil photo de votre iPhone et scannez le code QR situé à droite de cette page. Cliquez sur le lien jaune qui apparaît à l'écran.

3

Installer VDF Ambulance

TestFlight va s'ouvrir automatiquement. Cliquez simplement sur "Accepter" puis sur "Installer". L'application VDF apparaîtra sur votre écran d'accueil.

Bientôt sur l'App Store (Officiel)

L'application sera disponible publiquement sur l'Apple App Store d'ici 1 semaine (autour du 16 Avril 2026), sous réserve des délais de validation par Apple. L'installation TestFlight ne sera alors plus nécessaire.

Installation pour Android

Via l'accès direct Web / Progressive Web App (PWA).

1

Ouvrir le portail via Chrome

Ouvrez l'application Google Chrome et allez sur le site sécurisé de l'application :
👉 dev.vdf-ambulance.fr

2

Ouvrir les Options Chrome

Une fois sur la page de connexion bleue, cliquez sur les 3 petits points (⋮) situés tout en haut à droite de l'écran.

3

Installer l'Application 👆

Dans le menu qui s'ouvre, cliquez sur l'option :
Ajouter à l'écran d'accueil ou Installer l'application
Confirmez pour retrouver l'icône VDF avec vos autres applications !

Portail Android (Chrome)

Notifications Push

Au premier démarrage, l'application vous demandera l'autorisation d'envoyer des notifications. Cliquez obligatoirement sur Autoriser pour recevoir vos courses en temps réel.
